## DSL Design

- `@title`: lesson title
- `@beginq` and `@endq`: marks the start/end of a question block
- `@code`: start of code block (terminated by `@endcode`)
- `\`inline code\``: Wrap inline code in backticks
- `@text`: optional static text block (paragraph, note, etc.)
